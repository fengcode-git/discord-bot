# discord 机器人

执行环境：需要 Node.js 16.9.0 或更高版本

数据库：mongodb

使用说明：

1. 先在[https://discordapp.com/developers](https://discordapp.com/developers)网站获取机器人的TOKEN.

2. 在项目根目录创建文件.env，其内容参考example.env文件

3. 打开机器人的URL，加入到服务器

Yarn命令：

- dev：以开发模式运行
- build：编译项目
- start：以生产模式运行

机器人命令：

- /vote 数字：进行投票，参数为预测价格
- /query 数字：查询最接近真实价格的投票，参数为真实价格
